# Курсовой проект от аккадемии TOP .NET на c#

Тема: Создание чата для общения по локальной сети между компьютерами

ТЗ:
КЛИЕНТ:
клиент подразумевает консольное приложение для ввода и окном для получения информации, может соединяться локальным сервером и отправлять на него запросы

СЕРВЕР:
Сервер умеет принимать запросы отправленные клиентами и сохранять их в текстовый фаил(chat.txt) а также по запросу клиента отправлять данные из фаила клиенту

1. создание клиента на c# для связи и общения с сервером

2. создание сервера для получения хранения и отправки сообщений от клиентов

3. проверка соединения между клиентом и сервером

4. тест связи между локальными компьютерами аккадемии TOP

НАВИГАЦИЯ:
для управления клиентом используеться всего 2 команды
отправить - оправка сообщений на сервер
получить - получить все сообщения хранящиеся на сервере
